## 🚀 Fonctionnalités

- 🎨 Modification des icônes (`icon0`)
- 🖼️ Modification des backgrounds (`pic1`)
- 🧩 Custom du background XMB

👉 Glissez n’importe quelle image, le tool gère tout automatiquement :
- Conversion
- Redimensionnement
- Renommage
- Placement
